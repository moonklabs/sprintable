# Placeholder Image Guide

After first deployment, replace the placeholder screenshots in `docs/screenshots/` with actual ones:

1. Run `pnpm dev` and open `http://localhost:3108`
2. Capture the following pages:
   - Kanban board → save as `docs/screenshots/kanban.svg` (or `.png`)
   - Meeting notes → save as `docs/screenshots/meetings.svg`
   - Pricing page → save as `docs/screenshots/pricing.svg`
3. Commit and push — the README hero images update automatically.
